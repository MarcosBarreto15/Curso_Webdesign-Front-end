Padding

A propriedade padding define uma distância entre o conteúdo de um elemento e suas bordas. É um atalho que evita definir uma distância para cada lado separadamente: paddin-top, padding-right, padding bottonm e padding-left. Aqui nesse parágrafo foi criado uma classe de mesmo nome, com valores definidos no css.

Margin

A propriedade margin no CSS define a area da margem dons quatro lados do elemeno. É uma abreviação que define todas as margens individuais de uma só vez: margin-top, margin-righ, margin-botton e margin left. Nesse paragrafo foi criado uma classe de mesmo nome, com valores definidos no css.

Border

A propriedade CSS abreviada border define a borda de um elemento. Ele define os valores de largura da borda, estilo da borda e cor da borda (border-width, border-style, and border-color). Aqui foi criado uma classe borer para testar , com valores definidos no css.

Outline

A propriedade CSS outline é uma propriedade abreviada para configurar uma ou mais das propriedades de contorno outline-style, outline-width e outline-color em uma única declaração. Na maioria dos cados, o uso abreviado é preferével e mais conveniente

Contornos se diferenciam de bordas das seguintes maneiras:

  • Contornos não ocupam espaços, eles são desenhados acima do conteúdo.
